# NetLibrary
Bu proje, C# ve .NET teknolojileri kullanılarak geliştirilmiş, terminal (konsol) üzerinde çalışan hafif yapılı bir kütüphane yönetim sistemidir. Veritabanı bağlantısı gerektirmeden, kod içerisine gömülü (hardcoded) kitap ve üye verileriyle çalışır. Kullanıcılar, kitap ve üye numaralarını (ID) kullanarak temel kütüphanecilik işlemlerini simüle edebilirler.
